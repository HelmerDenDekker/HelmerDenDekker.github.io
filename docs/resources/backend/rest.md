## The REST application architecture

REST is an acronym for Representational State Transfer and an architectural style for distributed hypermedia systems.

## Constraints of REST architecture

1. Uniform interface
2. Client-Server
3. Stateless
4. Cacheable
5. Layered system
6. Code on demand
